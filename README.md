# WhatsApp Automation API

A modern Next.js application that demonstrates WhatsApp Cloud API integration, allowing users to send approved WhatsApp template messages, test message delivery, and manage customer communication workflows through the Meta WhatsApp Business Platform.

## Features

* WhatsApp Cloud API integration
* Send approved template messages
* Appointment confirmation messaging
* Responsive user interface
* Environment-based configuration
* Built with Next.js and React
* Ready for deployment on Vercel

## Tech Stack

* Next.js
* React
* TypeScript / JavaScript
* Tailwind CSS
* WhatsApp Cloud API
* Meta Developers Platform

## Getting Started

### Clone the Repository

```bash
git clone https://github.com/your-username/whatsapp-automation-api.git
cd whatsapp-automation-api
```

### Install Dependencies

```bash
npm install
```

### Configure Environment Variables

Create a `.env.local` file:

```env
NEXT_PUBLIC_API_URL=your_backend_api_url
NEXT_PUBLIC_PHONE_NUMBER_ID=your_phone_number_id
```

### Run Development Server

```bash
npm run dev
```

Open:

```text
http://localhost:3000
```
